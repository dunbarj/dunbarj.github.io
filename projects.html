Smart-Cloud Project

The Smart-Cloud is a physical lamp that was created to look like a cloud. It is small (about a foot in diameter) and contains a Raspberry Pi and a strip of LEDs. This self-started project involves creating a web server that a user can interact with to change the functionality and color of the cloud. The cloud can currently represent weather data using its RGB LED strip by changing color based on the temperature. Soon it will be able to show the forecast (rain/snow/shine) and eventually be able to be voice controlled with an Amazon Echo Dot using Alexa.
